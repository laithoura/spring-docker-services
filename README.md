# Spring Docker Microservices

It's a docker-based microservices application.

# Build Tool

- Maven 3.5.*

# Prerequisite
- Docker
- Docker Compose
- OpenJDK 8
- Spring Boot 2.5.7
- Lombok
- ...

# Start

Start containers without logs
```sh
docker-compose up -d
```

Stop containers:
```sh
docker-compose stop
```

Start containers
```sh
docker-compose start
```

Stop & Destroy all containers
```sh
docker-compose down
```
